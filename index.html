Type /track followed by any channel name, handle, URL, or ID. Autocomplete handles the rest.

02

Choose where updates post.

Use channel_output to send updates to a dedicated stats channel instead of the command channel.

Command /track channel: @creator channel_output: #milestones
Result

All subscriber embeds for that creator land in the selected Discord channel.

Check growth with /fastest.
